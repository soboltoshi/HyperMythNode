# MediaSurfaceBox

**IS:** The sovereign block that owns browser/public text-shell UI, chart surfaces, livestream overlays, AI video generation policy, and publication rules.

**OWNS:**
- public_text_style_rules
- button_style_rules
- chart_surface_state
- overlay_state
- ai_video_policy
- video_job_registry
- clip_registry
- publication_log
- public_manifest_state

**MAY:**
- Render browser and public text-shell surfaces
- Render chart overlays and livestream takeovers
- Request video jobs through HyperCinemaAdapter
- Let browser/mobile/chatbot users request and receive generated videos
- Read provider/model policy from ModelSovereigntyBox
- Publish videos, clips, and reports to approved surfaces
- Render parody overlay policy for webcam mode

**MAY NOT:**
- Let HyperCinemaAdapter own publication truth
- Let private-brain outputs publish directly
- Rewrite treasury, identity, or governance truth
- Bypass PublicReadiness when a surface becomes public default

**CONNECTS TO:**
- SessionKernelBox
- TianezhaSuperBrainBox
- ModelSovereigntyBox
- WorldDirectoryBox
- ArchivistBox
- PublicReadinessBox
