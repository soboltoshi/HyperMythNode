# Operator Setup Guide

## Tools to install
- Node.js 22+
- pnpm
- Git
- VS Code
- Firebase CLI

## Commands
```bash
pnpm install
pnpm dev
```

## Gotchas
- Keep secrets server-side
- Do not put Helius/Jupiter logic in core boxes
- Do not start with full voxel runtime; start with world activation and finance/media shell
