# Amendment: Add AssemblyAssimilationBox

## Accepted change

The Interface Assembly world gains a new sovereign box:

- `AssemblyAssimilationBox`

## Structural effect

This box becomes the native translator for foreign repos, folders, services, and model stacks.

It emits normalized adoption material but does not override WorldBuilder, InterfaceRegistry, Arbiter, Archivist, or BlockBuilder.

## New native URLs

- `INTERFACE://SKILL/ASSIMILATION/REPO`
- `INTERFACE://SKILL/ASSIMILATION/GITHUB`
- `INTERFACE://SKILL/ASSIMILATION/BOXMAP`
- `INTERFACE://BOX/ASSEMBLY-ASSIMILATION`
- `INTERFACE://WORLD/ASSIMILATION-CONTRACT`

## New verbs

- `ASSIMILATE://GITHUB+REPO`
- `ASSIMILATE://LOCAL+FOLDER`
- `ASSIMILATE://SERVICE+API`
- `ASSIMILATE://MODEL+STACK`
- `EMIT://BOXMAP`
- `EMIT://RUNTIMECONTRACT`
- `EMIT://MESHROLE`
- `EMIT://RISKREGISTER`
- `EMIT://INTERFACEURLS`

## Exclusion

No external repo is bundled into the core of this amendment. MiroFish and any other foreign system remain adapter candidates to be handled through the new assimilation flow.
