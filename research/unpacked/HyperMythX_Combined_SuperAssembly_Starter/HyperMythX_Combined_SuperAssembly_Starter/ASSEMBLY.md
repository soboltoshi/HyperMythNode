
# ASSEMBLY

HyperMythX is assembled from three locked layers:

1. **Constitution layer**
   - charter
   - constitution
   - block coding rules
   - world clauses
2. **Interface Assembly layer**
   - boxes
   - interfaces
   - adapters
   - starter prompts
3. **SuperAssembly layer**
   - runnable services
   - registry code
   - domain shells
   - supernode bootstrap
   - ASIMOG governance shell

## Lock rules

- SessionKernel is the sole canonical writer.
- ASIMOG reviews block intent before expansion of any major box.
- VR Superbrain remains separate from desktop execution.
- Desktop Agent holds execution powers for wallets, devices, and runtime control.
- Adapter count is locked at 69 for this starter.
- New adapters require a formal proposal and registry bump.
