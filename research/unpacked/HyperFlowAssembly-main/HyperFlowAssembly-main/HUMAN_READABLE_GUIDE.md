# Human-readable guide

## What this repo pack does
This repository gives you a combined Interface Assembly for:
- 4-chain world activation
- token worlds
- sovereign non-token worlds
- compute markets
- public/private brain split
- Windows-only desktop embodiment
- web/mobile/chatbot finance shells
- Hyperliquid derivative staging
- video generation everywhere

It now has two layers:
- `world/` is the constitutional layer
- `apps/`, `nodes/`, `packages/`, and `shared/` are the executable layer

## Big ideas
### 1. It is not a launchpad
Tokens launch elsewhere. Worlds activate here.

### 2. Worlds can exist without tokens
Communities, creators, or private groups can activate sovereign worlds without a token.

### 3. Useful compute is the economic center
CPI69 is public. MYTHIV42 is your private 42-agent mesh rate.

### 4. Public and private intelligence are separated
Tianezha is the only bridge between the private brain and sub-agents.

### 5. External systems stay as adapters
They extend the world. They do not own truth.

## How to read it
1. Start with `world/WORLD.md`.
2. Read `world/registries/runtime-owners.md`.
3. Read `world/registries/extraction-ledger.md`.
4. Open the matching runtime folder under `apps/`, `nodes/`, `packages/`, or `shared/`.
5. Run `npm run validate` to confirm the assembly is internally consistent.
