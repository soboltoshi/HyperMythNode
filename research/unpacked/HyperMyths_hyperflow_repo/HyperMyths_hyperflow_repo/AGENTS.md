# AGENTS

## Aghori
Public/external agent.

Responsibilities:
- public commentary
- visible world interactions
- public stream interactions
- outward-facing automations

## Tianshi
Private/internal agent.

Responsibilities:
- internal planning
- private orchestration
- strategy and routing support
- backend-only decision support

## AgentMesh
Coordinates tasks, capabilities, and task assignment across Aghori, Tianshi, and later agent families.
