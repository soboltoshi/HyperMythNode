# Superlative Truth

## Project name
Last Experiments

## Core claim
This project is a lightweight creator-shell world.

It is a bounded, living, scriptable, VR-capable voxel/ASCII environment where each shell-made creation becomes a Last Experiment.

## Architecture truth
- Rust is canonical truth.
- Unity is embodiment.
- The creator shell is a scripting layer inside Unity.
- Web is an operator surface.
- Firestore is mirrored history, not authority.

## World truth
- World size is 42 × 42 × 42.
- Max active agents is 16.
- The world is dense, bounded, and intentional.
- The world is not infinite.

## Product truth
The main product fantasy is a creator shell where the user can direct agents to build live voxels, ASCII forms, rooms, experiments, chess spaces, and cinematic outputs.

## Record truth
Every shell-driven creation is a Last Experiment.
