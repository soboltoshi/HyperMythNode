# Charter

## Purpose
Last Experiments exists to build a bounded creator-shell world where command, scripting, embodiment, and experimentation are unified in one operating environment.

## Mission
Build a small, dense, living world where creators and agents can make structures, behaviors, sessions, and experiments live.

## Principles
1. Truth must be singular.
2. Worlds must remain bounded.
3. Tools must feel direct.
4. Scripting must be expressive but safe.
5. Every serious creation becomes a Last Experiment.
6. Dense worlds beat bloated worlds.
7. The shell comes first.

## Constitutional stack
- Rust for truth
- Unity for embodiment
- Lua-style shell for creator scripting
- JS/Next.js for operator web surfaces
- Firebase App Hosting for web deployment
- Firestore for mirrored logs/history
