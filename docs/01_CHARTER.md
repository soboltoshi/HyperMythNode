# Charter

## Last Experiments

### 0. Purpose

Last Experiments exists to build a bounded creator-shell world where a human can direct agents, inspect outcomes, and grow living systems without surrendering truth to spectacle, mood, vendor surfaces, or leaked private cognition.

This world is:

- desktop-first
- VR-capable
- creator-shell-first
- bounded
- agent-directed
- inspectable
- modular

### 1. Mission

Build a small, dense, living world where creators, routers, and agents can make structures, behaviors, rooms, experiments, and media outputs live, while preserving a singular truth engine and visible receipts for every serious action.

### 2. Why This Charter Exists

This charter exists because vague systems decay.

It exists:

- because unclear roles create resentment
- because authority drifts when ownership is not explicit
- because shells and mirrors become dangerous when they are mistaken for truth
- because public explanation and private synthesis must not collapse into one voice
- because desktop agents need guide rails before interpretation
- because agents need strong boundaries if they are to be useful without becoming sovereign
- because constitutions must be able to grow within limits

### 3. Core Principles

1. Truth must be singular.
2. Worlds must remain bounded.
3. Tools must be direct.
4. Interfaces must stay clean.
5. Every serious creation becomes a Last Experiment.
6. Public explanation and private reasoning must not be confused.
7. All agents and sub-agents are Hermes-agent roles, not sovereign minds.
8. Receipts matter more than vibes.
9. Growth is allowed only within natural limits.
10. Emergent tools may begin rough, but they do not gain authority until ratified.
11. Money never enters the game shell or agent cluster.

### 4. Constitutional Stack

The stack is fixed:

- Rust for canonical truth
- Unity for embodiment
- Lua-style shell for creator scripting
- `claw-code-adapter` as the desktop command-shell wrapper around the Lua creator shell
- Next.js and TypeScript for operator and router surfaces
- Hermes-agent for all agent and sub-agent roles
- Firestore for mirrored history

Everything else is subordinate to this stack.

### 5. Named Roles

#### Human Creator

The human sets purpose, chooses direction, approves major changes, and uses the shell to direct the world.

#### Desktop Shell

The desktop shell is the wrapper and orchestrator that stands between embodiment and deeper interpretation.
Its runtime basis is `claw-code-adapter`.
It routes into the Lua creator shell rather than replacing it.

#### Public Brain Router

The public brain router explains, summarizes, guides, and shapes command intent for outward-facing surfaces.
It may host `G0DM0D3` as a cognition mode or module.
It is presentation-facing intelligence.

#### Private Brain Router

The private brain router synthesizes high-context reasoning, personal worldseed, internal plans, and sensitive design pressure.
It is inward-facing intelligence and the only router allowed to touch internal desktop-agent orchestration.

#### Asimov Guide Gate

Asimov stands between the human-facing shell path and the interpreter path.
It guides the human and gates requests before interpretation.

#### Interpreter

The interpreter translates human intent, knowledge, code, and shell-language requests into structured proposals.

#### Agents

Agents perform scoped work under explicit task packets.
They are Hermes workers, not sovereign minds.

#### ASIMOG

ASIMOG is the arbiter cluster.
Its full expansion is `Agentic Swarm Intelligence Modification Organization & Guidance & Grievance`.
It handles routing review, guidance, and grievance handling inside the Hermes mesh.

### 6. Role Boundary Summary

- The desktop shell routes and stages.
- The public brain router may speak outward.
- The private brain router may think inward and orchestrate inward.
- Asimov may guide and gate.
- The interpreter may translate into structured proposals.
- Hermes agents may execute bounded work.
- ASIMOG may arbitrate routing and grievance.
- Only the truth kernel may commit world truth.

No other role may claim that power.

### 7. World Promise

The world promises:

- a bounded space of `42 x 42 x 16`
- no more than `16` active agents
- a creator shell with visible commands
- a desktop wrapper that guides before interpreting
- live previews before commitment
- recorded experiment history
- receipts for committed actions
- separation between public speech and private cognition

### 8. What This World Rejects

This charter rejects:

- infinite-world drift
- hidden authority
- direct shell mutation of truth
- direct agent mutation of truth
- browser-owned truth
- Unity-owned truth
- Firestore-owned truth
- public-brain access to internal desktop agents
- money-handling shells
- money-handling agents
- unbounded adapter sprawl as day-one architecture

### 9. Charter Covenant

The covenant is simple:

- the human directs
- the desktop shell stages
- Asimov guides and gates
- the interpreter proposes structure
- the creator shell proposes commands
- the routers separate public from private cognition
- Hermes agents execute bounded work
- ASIMOG arbitrates grievance and routing
- the kernel ratifies
- the mirror reflects

That is the governing order of Last Experiments.
